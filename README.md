## How to run
### Frontend
#### To run
```
cd frontend
npm install
npm start
```
#### To test
```
cd frontend
npm test
```

### Backend
#### To run
```
// Firstly activate your virtualenv for python and install django.
cd backend
python manage.py runserver
```
#### To test
```
cd backend
make test
```
